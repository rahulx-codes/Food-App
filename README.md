# 🍔 Food App

A full-stack food ordering web application built using React, Tailwind CSS, and Node.js.

## 🚀 Features

- User Login & Signup
- Add to Cart functionality
- View Cart Items
- Total Price Calculation
- Responsive UI
- Frontend & Backend Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

## 📂 Project Structure

Food-App
├── Frontend
└── Backend
